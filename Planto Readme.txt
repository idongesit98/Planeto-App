
# Planeto App

A planetary app that displays the nine planets using a recycler view and gives a brief description of each planet.


## Features 

- Recycler View 
- Glide for easy loading of pictures 
- Navigation Using Nav graph 
- 


## Lessons Learned
- How to build a recycler view with multiple item 
- Using glide to make images load quickly.
- Using Drawables to give a custom design to card views.
- Navigating from one screen to another through a recycler View.





## Language Used

**Language:** XML, Kotlin.
